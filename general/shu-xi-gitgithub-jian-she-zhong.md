## 2. Git 命令

git 是分布式的版本控制软件

### 2.1 配置账号和邮箱

设置账号和邮箱

```shell
git config --global user.name xxx
git config --global user.email xxx
```

长期存储密码

```shell
git config --global credential.helper store
```
